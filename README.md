🏦 Banking System (OOP Based)

📌 Overview
This project is a **Banking System application** developed using Object-Oriented Programming concepts. It allows users to perform core banking operations such as account creation, deposit, withdrawal, and balance inquiry.
The project demonstrates strong understanding of **OOP principles**, **data handling**, and **full-stack integration**

🚀 Features
* 🆕 Create new bank account
* 💰 Deposit money
* 💸 Withdraw money
* 📊 Check account balance
* 📋 View all accounts
* 🔄 Transaction handling system

🛠️ Tech Stack
* **Frontend**: HTML, CSS, JavaScript *(if used)*
* **Backend**: Node.js, Express *(if used)*
* **Database**: MySQL / File Handling *(based on your implementation)*
* **Languages**: C / C++

🧠 OOP Concepts Used
* **Encapsulation** → Secured account data and operations
* **Abstraction** → Simplified user interactions
* **Inheritance** → Different account types (if implemented)
* **Polymorphism** → Flexible transaction handling

⚙️ How to Run

🔹 For C/C++ Version
1. Compile the code:
   g++ main.cpp -o banking
2. Run:
   ./banking

🔹 For Web Version (Node.js)
1. Install dependencies:
   npm install
2. Start server:
   node app.js
3. Open in browser:
   http://localhost:3000
